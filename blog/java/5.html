# Gradle 发布到远程仓库

本文档使用 Gradle 的 maven 插件实现，详情请查看 maven 插件的官方文档

# 添加插件

在 plugins 的闭包中添加 id 'maven'。例如：

plugins {
    id 'maven'
}
1
2
3

# 发布到远程仓库

在 build.gradle 文件中添加如下代码：

uploadArchives {
    repositories {
        mavenDeployer {
            repository(url: '[你的仓库地址]') {
                authentication(userName: "[你的用户名]", password: "[你的密码]")
            }
        }
    }
}
